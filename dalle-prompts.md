# Gemini Nano Banana Image Generation Prompts for The Misty Cup Coffee Room

**API:** Google Gemini 2.5 Flash Image (Nano Banana)
**Aspect Ratio:** 3:2 landscape (1536x1024 equivalent)

## Instructions:

### Option 1: Automated Generation (Recommended)
Once Gemini API quota resets, use the updated `generate-tmc-images.js` script to automatically generate all images.

### Option 2: Manual Generation via Google AI Studio
1. Visit: https://aistudio.google.com/
2. Select "Gemini 2.5 Flash Image" model
3. Copy each prompt below
4. Set aspect ratio to 3:2
5. Generate and download the image
6. Rename to match the filename shown
7. Save to `public/images/tmc-item-images/`

### Option 3: Browser-based Puter.js (if available)
Use the Puter.js Nano Banana API in a browser environment if you have access.

**Note:** Gemini 2.5 Flash Image has free tier quota limits. Monitor usage at: https://ai.dev/usage?tab=rate-limit

### Via API (Automated):
1. **Update generate-tmc-images.js** to use Gemini endpoint
2. **Set rate limiting:** Gemini free tier has request limits
3. **Handle quota errors:** Script should gracefully handle 429 errors
4. **Batch processing:** Generate in smaller batches to stay within limits
